# 🍸 VODKA TIME - Countdown Page

> Eine trashige 90er-Webseite für den ukrainischen Vodka-Abend

## Features

- ⏱️ **Live Countdown** bis 15:30 Uhr
- 🎨 **Retro 90er Ästhetik** - Neon, Blink, Marquee, Windows 95 Dialoge
- 🌟 **Animiertes Sternenfeld** im Hintergrund
- 🍾 **Vodka-Guide** mit ukrainischen Empfehlungen
- 🎵 **Ukrainische Klassische Musik** via YouTube IFrame API (Play/Pause Button)
- 📋 **Teilnehmerliste** mit Live-Status
- 📱 **Responsive** - funktioniert auf Mobile
- 💻 Fake-Besucherzähler wie früher 👁️

## Deploy auf GitHub Pages

```bash
git init
git add .
git commit -m "🍸 Vodka Countdown - Za zdorovya!"
git branch -M main
git remote add origin https://github.com/DEIN_USERNAME/vodka-countdown.git
git push -u origin main
```

Dann unter **Settings → Pages → Source: main / root** aktivieren.
